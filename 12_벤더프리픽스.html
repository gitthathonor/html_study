벤더프리픽스(vendor prefix)

각 브라우저마다 css3 속성중 표준안으로 확정되지 않은 속성이 있음. 이때 해당 브라우저 접두어를 붙여야 각 개별 브라우저별로 대응함 -webkit- : 크롬, 사파리, 안드로이드, 아이폰 브라우저 적용 -moz- : 파이어폭스, 모질라 브라우저 -ms- : IE, 엣지 -o- : 오페라 브라우저

ex) #box{ -webkit-border-radius: 10px;image.png -moz-border-radius: 10px; -ms-border-radius: 10px; -o-border-radius: 10px; border-radius: 10px; }
