1) background-color 속성 : 배경색상만
d2
2) background-image : 배경 이미지만 {background-image:url(경로)} => 기본적으로 w와 h의 공간이 있어야 정확히 배경이미지가 출력됨 => 기본적으로 w,h 무한반복해서 이미지 출력
d3
3) background-repeat : 배경이미지 반복여부설정
값 : no-repeat
repeat-x
repeat-y
d4
4) background-position: 배경이미지 출력 위치 - 기본 (0,0) 왼쪽 상단 - 배경이미지가 반복하고 있다면 {background-position} 쓸 수 없음 {background-repeat:no-repeat} 일 때 함께 사용
d5
5) background속성 : 1~4번의 개별속성을 한 번에 처리할 때 쓸 수 있는 속성 {background:색상 url() 반복여부 가로 세로;} * 값을 순서대로 작성하는 것이 중요 * 4가지 속성값을 모두 작성해야 되는 것이 아님
d6
6) background-size : 배경이미지 크기 조절 {background-size:가로크기 세로크기} => {background-size:150px auto;} : 가로크기 150px, 세로크기는 auto=자동(이미지비율) 에 맞춰서 알아서 인식해라
d7
7) background-attachment 속성 : 배경이미지의 스크롤방법을 지정 하는 속성 {background-attachment:fixed} : 해당 태그의 영역에서만 배경이미지가 출력 웹 브라우저 기준 배경이미지를 고정시킴
