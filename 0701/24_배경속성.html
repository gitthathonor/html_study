d1
1) background-color 속성 : 배경색상만
d2
2) background-image : 배경 이미지만 {background-image:url(경로)} => 기본적으로 w와 h의 공간이 있어야 정확히 배경이미지가 출력됨. => 기본적으로 w,h 무한반복해서 이미지 출력.
d3
3) background-repeat : 배경이미지 반복여부설정 값 : no-repeat repeat-x repeat-y
d4
4) background-position: 배경이미지 출력 위치 - 기본 (0,0) 왼쪽 상단. - 배경이미지가 반복하고 있다면 {background-position} 쓸수없음. {background-repeat:no-repeat} 일때 함께사용.
d5
5) background속성 : 1~4번의 개별속성을 한번에 처리할때 쓸수있는 속성. {background:색상 url() 반복여부 가로위치 세로위치;} * 값을 순서대로 작성하는 것이 중요. * 4가지 속성값을 모두 작성해야 되는것이 아님.
d6
6) background-size : 배경이미지 크기 조절 {background-size:가로크기 세로크기} => {background-size:150px auto;} : 가로크기150, 세로크기는 auto=자동 (이미지비율) 에 맞춰서 알아서 인식해라.
d7
7) background-attachment 속성 : 배경이미지의 스크롤방법을 지정 하는 속성. {background-attachment:fixed} : 해당태그의 영역에서만 배경이미지가 출력. 웹브라우저 기준 배경이미지를 고정시킴.





css 이미지 스프라이트 기법 (image sprite)

  1. 여러개의 이미지를 하나의 이미지로 합쳐서 사용하는 방법. => 이미지의 관리 용이 / 웹사이트 이미지 로딩시 시간 및 용량을 줄일 수 있음. => 이미지 적용시 웹표준을 준수해야 하므로 가상대체 텍스트 를 사용해야함. (IR: Image Replacement 기법사용)
