* HTML 이란?
- 웹브라우저가 알아들을수 있는 문서의 형식(html문서)
- html문서는 태그 <tag>로 구성됨.
- 태그란? html문서를 구성하는 명령어
- <태그명></태그명>
- 태그작성시 소문자 사용

=================

*html문서의 기본형식
<html>
     <head>
        <title>웹문서제목</title>
     </head>
     <body>
        웹브라우저에 출력될 내용 영역
     </body>
</html>

=================

*html문서내에서는 엔터키, 스페이스바는 띄어쓰기 한번만 인식함.
강제 줄바꿈 하기 위해서는 <br>태그 사용해야함.

스페이스바(공백)삽입하기 위해서는 특수기호 &nbsp; 사용해야함.

=================
*주석문
코드에 작성하지만, 웹브라우저 상에 출력되지 않는 영역.
html, css문서의 종류에 따라 주석의 종류도 달라짐.
주석문 안에 또다른 주석문 포함되어지면 오류남.
html문서에 css주석 적으면 오류 / css문서안에 html주석 작성시 오류

주석의 형태 2가지
<!--html주석문 -->
/* css주석문 */

=================
*기본태그
<div></div>                : 웹 ui의 영억을 나누는 역할(블록요소)
<p></p>                    : 웹 ui의 영역안의 문단(단락) 나누는 역할(블록요소)
<h1></h1>~<h6></h6> : 제목태그(블록요소)
<br>                          : 줄바꿈 역할
